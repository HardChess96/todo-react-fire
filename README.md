# todo-react-fire
- React, firebase ve blueprintjs kullanarak gerçekleştirdiğim basit bir to-do uygulaması gerçekleştirirken yaptığım adımları medium'a yazdım.

## Çalıştırmak İçin 

```
$ npm i --save 
-> firebase config ayarlarınız için düzenleyin. config.js
$ npm start
```
